Tecnologias

Python

Python é uma linguagem dinâmicamente tipada, bastante utilizada nos dias atuais. Como desenvolvedor, aplico para construção de automações de tarefas e também tratamentos de dados com bibliotecas como Pandas e Numpy.

Django

Django é um framework em python, uma ferramenta bastante útil quando se trata de construção de aplicações web de forma rápida. Além disso, o Django tem seu destaque na construção de APIs, bastante essenciais para comunicação entre aplicações.

Flutter Flow

Flutter Flow é uma ferramenta no-code online, que é utilizada para construção de aplicações web e mobile de forma mais rápida. Sua interface oferece bastante recursos para construção de aplicações, seja para testes ou publicação na loja.

Bubble

Bubble é uma ferramenta no-code online, voltada para construção de aplicações web. Sua interface permite a construção rápida de páginas, alimentados por um banco local ou externo. Sua plataforma também oferece suporte para o consumo de APIs externas ou plugins.

Projetos

XPLS

Este foi um dos primeiros programas de software que desenvolvi. Seu objetivo era claro: otimizar o processo de montagem de planilhas para o plano de saúde da empresa. Através de um arquivo PDF recebido, o programa era capaz de extrair e organizar automaticamente todos os dados necessários em uma planilha, tornando o processo mais ágil e preciso. O programa foi feito em Python, utilizando principalmente bibliotecas como Pandas e Numpy.

Fourmaq Connect

Fourmaq Connect é uma solução inovadora desenvolvida para resolver um desafio comum na empresa. O projeto foi focado na criação de um sistema de reservas de máquinas, permitindo que os usuários realizem reservas diretamente pelo celular. Toda a gestão e controle dos pedidos são feitos de maneira ágil e eficiente pela plataforma web, garantindo um processo mais organizado e acessível para todos os envolvidos. Essa aplicação foi contruída em Python, utilizando do framework Django, na construção de APIs, além de plataformas como Flutter Flow, para desenvolvimento Mobile e Bubble para desenvolvimento Web.
